## 💡 使用技巧

### 技巧 1: 使用搜索

所有文档都是Markdown格式，可以：
- 在编辑器中使用 Ctrl+F / Cmd+F 搜索
- 使用 `grep` 命令行搜索：
  ```bash
  grep -r "关键词" docs/
  ```

### 技巧 2: 标记重要内容

阅读时标记重要部分：
- ⭐ 必读部分
- ⚠️ 注意事项
- 💡 有用的技巧
- 🔍 需要深入研究

### 技巧 3: 渐进式学习

不要一次读完所有文档：
1. 先快速浏览，了解结构
2. 根据需要深入特定章节
3. 边学边实践
4. 遇到问题时回来查阅

### 技巧 4: 做笔记

创建自己的笔记：
```bash
cd /root/ms/DiffMS/docs
nano MY_NOTES.md
```
